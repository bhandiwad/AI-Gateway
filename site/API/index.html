    ðŸ“¡ API Reference¶

    Complete API documentation for AI Gateway. All endpoints are prefixed with /api/v1.

    Authentication¶

    Register New Tenant¶

    POST /api/v1/admin/auth/register
Content-Type: application/json

{
  "name": "Your Name",
  "email": "you@example.com",
  "password": "SecurePassword123!"
}

    Response (200 OK): 

    {
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer",
  "tenant": {
    "id": 1,
    "name": "Your Name",
    "email": "you@example.com",
    "is_active": true,
    "is_admin": false,
    "rate_limit": 100,
    "monthly_budget": 100.0,
    "current_spend": 0.0
  }
}

    Chat Completions¶

    OpenAI-compatible chat completion API.

    Create Chat Completion¶

    POST /api/v1/chat/completions
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "model": "gpt-4o-mini",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Hello!"}
  ],
  "temperature": 0.7,
  "max_tokens": 1000,
  "stream": false
}

    Parameters:

    Parameter Type Required Description
    model string Yes Model ID (e.g., "gpt-4", "claude-3-opus")
    messages array Yes Array of message objects
    temperature float No Sampling temperature (0-2)
    max_tokens integer No Maximum tokens to generate
    top_p float No Nucleus sampling parameter
    stream boolean No Enable streaming response
    stop array No Stop sequences
    presence_penalty float No Presence penalty (-2 to 2)
    frequency_penalty float No Frequency penalty (-2 to 2)

    Response (200 OK): 

    {
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "created": 1677858242,
  "model": "gpt-4o-mini",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "Hello! How can I help you today?"
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 20,
    "completion_tokens": 10,
    "total_tokens": 30
  }
}

    Streaming Response¶

    With stream: true, response is Server-Sent Events:

    data: {"id":"chatcmpl-abc123","choices":[{"delta":{"content":"Hello"}}]}

data: {"id":"chatcmpl-abc123","choices":[{"delta":{"content":"!"}}]}

data: [DONE]

    API Keys¶

    Create API Key¶

    POST /api/v1/admin/api-keys
Authorization: Bearer YOUR_TOKEN
Content-Type: application/json

{
  "name": "Production Key",
  "description": "Key for production environment",
  "environment": "production",
  "rate_limit_override": 500,
  "allowed_models_override": ["gpt-4", "gpt-3.5-turbo"],
  "cost_limit_daily": 50.0,
  "cost_limit_monthly": 500.0
}

    Response (200 OK): 

    {
  "id": 1,
  "name": "Production Key",
  "api_key": "sk-gw-abc123...",
  "key_prefix": "sk-gw-abc",
  "is_active": true,
  "environment": "production",
  "rate_limit_override": 500,
  "created_at": "2024-01-15T10:30:00Z"
}

    List API Keys¶

    GET /api/v1/admin/api-keys
Authorization: Bearer YOUR_TOKEN

    Response (200 OK): 

    [
  {
    "id": 1,
    "name": "Production Key",
    "key_prefix": "sk-gw-abc",
    "is_active": true,
    "environment": "production",
    "last_used_at": "2024-01-15T12:00:00Z",
    "created_at": "2024-01-15T10:30:00Z"
  }
]

    Revoke API Key¶

    DELETE /api/v1/admin/api-keys/{key_id}
Authorization: Bearer YOUR_TOKEN

    Response (200 OK): 

    {
  "message": "API key revoked successfully"
}

    Audit Logs¶

    List Audit Logs¶

    GET /api/v1/admin/audit-logs
Authorization: Bearer YOUR_TOKEN

    Query Parameters:

    Parameter Type Description
    action string Filter by action type
    start_date string Start date (ISO format)
    end_date string End date (ISO format)
    limit integer Max results (default: 100)
    offset integer Pagination offset

    SSO¶

    List SSO Providers¶

    GET /api/v1/admin/auth/sso/providers

    OIDC Discovery¶

    POST /api/v1/admin/sso/discover
Authorization: Bearer YOUR_TOKEN
Content-Type: application/json

{
  "issuer_url": "https://accounts.google.com"
}

    Initiate SSO Login¶

    POST /api/v1/admin/auth/sso/initiate
Content-Type: application/json

{
  "provider_name": "google",
  "redirect_uri": "https://your-app.com/callback"
}

    SSO Callback¶

    GET /api/v1/admin/auth/sso/callback?code=AUTH_CODE&state=STATE&provider_name=google

    Error Responses¶

    All errors follow this format:

    {
  "detail": "Error message description"
}

    401 Unauthorized
    403 Forbidden
    404 Not Found
    422 Validation Error
    429 Rate Limited
    500 Internal Server Error
